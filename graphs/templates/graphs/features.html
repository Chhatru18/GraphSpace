GraphSpace Features

Interactive graphs

  • Upload and view interactive graphs via REST API or User Interface
  • Save and customize layouts of graphs
  • Search for elements inside of graphs
  • Sequentially step through subgraphs of the entire graph

Share graphs with collaborators

  • Share graphs between all members of a group
  • Interact and manipulate other group members' graphs
  • Create and save layouts for group members' graphs

Organization of graphs

  • Tags allow for organizing multiple graphs
  • Quickly search for all graphs that match tag

REST API

  • Programmatically communicate with GraphSpace
  • Manage graphs and faciliate graph uploads
  • Allows one to easily create scripts to interact with features of GraphSpace
