This tutorial will teach you how to upload a simple graph into GraphSpace. First of all, you will need an account, which you can create here, as well as a Unix machine. You will need 

curl
installed on the machine, which can be installed by running 
 sudo-apt-get install curl
if it is a Debian based machine such as Ubuntu.

We will use a graph with two nodes and one edge connecting the nodes in this tutorial. GraphSpace uses a JSON format to store graphs. The JSON code representing the graph is shown below.

{
    "graph": {
        "nodes": [
            {
                "data": {
                    "shape": "ellipse",
                    "id":"P43146",
                    "label": "DCC"
                }
            }, 
            {
                "data": {
                    "shape": "ellipse",
                    "id":"P08107",
                    "label": "HSPA1B"
                }
            }
        ],
        "edges": [
            {
                "data": {
                    "id": "P43146-P08107",
                    "source": "P43146",
                    "target": "P08107",
                    "directed": "true",
                }
            }
        ]

    },
    "metadata": {
        "description": "This is a tutorial graph",
        "tags": [
            "tutorial"
        ]
    }
}

Save the code above into a file called 

graph.json
. Now, open a terminal and navigate to the location of the file. Execute the following command, where 
[username]
is the email address you used to sign up for GraphSpace, and 
[password]
is your GraphSpace password.

curl -X POST "{{url}}api/users/[email]/graph/add/tutorial/" -F "username=[email]" -F "password=[password]" -F graphname=@"graph.json"

Now go to your graphs page and you should see a graph called 'tutorial' in the list.

For a more detailed reference on all of the functions of the GraphSpace API, look at the REST API help page.
